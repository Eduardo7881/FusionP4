# Contributing to FusionP4

Thank you for your interest in contributing!

## How to Contribute

1. Fork the repository
2. Create a new branch:
   git checkout -b feature-name
3. Commit your changes
4. Push and open a Pull Request

## Code Guidelines

- Use C++ for core components
- Keep code clean and documented
- Follow consistent naming conventions

## Important Rules

- ❌ Do NOT include proprietary Sony code
- ❌ Do NOT include leaked SDKs or firmware
- ❌ Do NOT upload copyrighted assets

All contributions must be original and clean-room implemented.

## Reporting Issues

Please include:
- Logs
- System info
- Steps to reproduce
